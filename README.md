Поиск и сохранение аббревиатур и их расшифровок в PDF документах.

Выполняется по следующим паттернам:
1. Определение (АББР)
2. АББР — определение
3. Поиск в разделе с обозначениями и сокращениями 
4. АББР (Определение)
...

Документы помещаются в директорию /documents.
Аббревиатуры сохраняются в `data/abbreviations.json`.

Планируемые улучшения:
- очистка дубликатов в json и приведение к одному виду
